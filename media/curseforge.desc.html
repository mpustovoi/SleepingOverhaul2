ğŸ·ï¸ All Features & Config


The mod aims to be as configurable as possible, so here's a run-down of all features sorted by config location and order.


sleepingoverhaul-server.toml:

Note: This config generates under "serverconfig" in your world/save folder

[general]

sleepAction:
- The action to perform when all players are sleeping.
- Can be "Timelapse" (default), "SkipToDay" (vanilla) or "Nothing" (nothing will happen).

[morning]

morningResetWeather:
- Reset the weather on morning (when players wake) if raining. Default true (same as vanilla).

[timelapse]

sleepAttackedAction
- The action to perform on a player if they are attacked during timelapse sleep
- Can be "NoChange" (default), "InstantKill" or "Invincible"
sleepPreventMagicDamage
- If true (NOT default), damage type of "Magic" (pre-1.19.4) or "Indirect" (1.19.4+) will not apply to players during timelapse. This includes DoT's like Poison.
logTimelapsePerformanceStats:
- If true (default), will performance stats will be logged on timelapse end.
- Shows average TPS, total time and total ticks.

disableNaturalSpawning:
- If true, natural spawning will be disabled during timelapse.
- Gives a minor speed boost.

disableLivingEntityTravel:
- If true (NOT default), LivingEntity type mobs will not travel during timelapse.
- Gives a minor speed boost.
- Disabled by default since it could result in undesired loss, e.g. mobs drowning.
- Note that this does NOT include Villager movement.

[bedRest]

bedRestEnabled:
- Allows players to rest in a bed without sleeping, adding a 'Sleep' button next to 'Leave Bed'
- true by default
bedRestWithChatEnter:
- Allows pressing Enter with no chat text to activate Sleep while resting.
- Requires the above bedRestEnabled to be true to work.
- true by default.
